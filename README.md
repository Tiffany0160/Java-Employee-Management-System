# Employee-Management-System Java

A professional **Java project** demonstrating a complete **Employee Management System** using:

- Object-Oriented Programming (OOP)
- Backend service layer
- Frontend Console Menu UI
- CRUD operations with search functionality
- ArrayList-based database simulation (beginner-friendly, replaceable with MySQL later)

This project is perfect for showcasing a real developer portfolio, demonstrating clean structure, modular code, and standard Java project practices.

---

## 📂 Project Structure

EmployeeManagementSystem
│
├── Main.java
├── model
│ └── Employee.java
├── service
│ └── EmployeeService.java
└── ui
└── Menu.java

### ✅ Files Explained

- **Employee.java**: Model class representing an employee with attributes `id`, `name`, `position`, `salary` and methods to display and update data.  
- **EmployeeService.java**: Service layer handling business logic like adding, viewing, updating, deleting, and searching employees.  
- **Menu.java**: Console-based user interface allowing users to interact with the system.  
- **Main.java**: Program entry point that starts the console menu.  

---

## 🛠 Features

- Add new employees  
- View all employees
